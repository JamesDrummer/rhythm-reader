# Rhythm Reader

A browser-based drum rhythm reading game for BHDA students.

## Local development

```bash
npm install
npm run dev
```

Useful checks:

```bash
npm run lint
npm test
npm run build
```

This first milestone contains the React app shell, BHDA theme, routing, and
placeholder screens. The rhythm model, audio, notation, scoring, and exercise
content will be added in the later milestones defined in the product spec.
